
* cerebro-org-todo

Manage org mode todos in [[https://github.com/KELiON/cerebro][cerebro]].

* Installation

Install "Org Todo" plugin via Cerebro Plugin Manager.

* Usage

** Example

   [[./example.gif]]

** Settings

Configure TODO file under "Manage Plugins" in Cerebro.

* Development

** Install dependencies

#+begin_src sh
yarn
#+end_src

or

#+begin_src sh
npm install
#+end_src

** Link application

#+begin_src sh
yarn start
#+end_src

or

#+begin_src sh
npm run start
#+end_src
